ID card (KYC)
This is today the most secured way to bind a blockchain address to a user identity for TFR regulation as the user cannot switch between different devices. The KYC process is carried out by a KYC provider with a 3D facial liveness test and document authentication. Altme keeps the KYC provider Id file for audit. The KYC Id file, the identity private key and blockchain address private key are bound together.
