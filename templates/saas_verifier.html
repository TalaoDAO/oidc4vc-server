Welcome to the Self Sovereign Identity verifier generator.

One can now authenticate a user who carries her own credentials in her Self Sovereign Identity wallet in any type of applications.

How it works ?


For each user for whom you want to authenticate with credentials, your application requests an OpenID authentication (code flow or implicit flow).
The platform will manage a user redirect to a page with a QR code (or deeplink for mobile device) allowing users to present a verifiable credential-presentation stored in their wallet.
Your app will receive the user credential as an Id token or a json file on the userinfo endpoint.
The verification of the signatures of the credential issuer and the user's wallet is carried out by the platform, your application can simply access the attributes of the json received in complete safety.
You can now update or generate different verifiers for your needs. The code of this platform is open source.
The platform supports SSO for CMS as Webflow and others (Wordpress, Drupal, Joomia, Magento, Shopify,....NodeJs, Python,...) through MiniOrange.
