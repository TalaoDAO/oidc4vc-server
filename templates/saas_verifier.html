Welcome to the verifier configurator and generator.



You can add a verifier to your code as a login similar to Facebook or Google Connect (or FranceConnect).


What you need to do is to install in your code a client OpenID library.
It takes no more than 2 hours.


A list of libraries available for different platforms is here.


How it works ?


For each user for whom you want to obtain verifiable credentials or authenticate with credentials, your applications requests OpenID authentication.
The platform will manage a user redirect to a page with a QR code (or deep link for mobile browser) allowing users to present a verifiable credential-presentation stored in their wallet.
Your app will receive the user credential as a json file on the userinfo endpoint.
The verification of the signatures of the issuer and the user's wallet is carried out by the platform, your application can simply access the attributes of the json received in complete safety.
You can now update or generate different verifiers for your needs. The code of this platform is open source.
