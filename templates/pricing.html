DeFi : best KYC AML process for user onboarding


It is today the most secure way to bind a blockchain address to a natural person (TFR, ...).
The KYC process is carried out through a 3D facial liveness test and an integrated document authentication process in the mobile device with prior biometric authentication of the user. The user can associate multiple blockchain addresses to the same decentralized identity in one step. Privacy (GDPR : right to be forgotten) is guaranteed because the private key of the decentralized identity is different from the private key of the blockchain address.
Altme retains the KYC process ID for auditing. KYC data, the decentralized identity and blockchain addresses are bound by cross-signatures.
